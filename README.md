## Description

Pinterlist is a web application that turns Pinterest recipes into a grocery list.

Stack: MERN, Redux, Bootstrap 4

Deployment: Heroku

## Pinterlist in action!

![](Pinterlist-demo.gif)

## Usage

```bash
npm run dev
```

Pinterlist will then be live on localhost 3000
